# GLRP Free Website

A free static website for Gang Life RP. It works with GitHub Pages, Cloudflare Pages or Netlify.

- YouTube: `https://www.youtube.com/@GLRPFiveM`

The TikTok link remains optional and can be added later inside `config.js`.

## Test on your computer

Open `index.html` in a browser. For the most accurate test, use VS Code with the free Live Server extension.

## Publish free with GitHub Pages

1. Create a new public GitHub repository, for example `glrp-website`.
2. Upload every file and folder from this package.
3. Open the repository Settings.
4. Select Pages.
5. Under Build and deployment, choose `Deploy from a branch`.
6. Choose branch `main`, folder `/ (root)`, then Save.
7. GitHub will display your free live URL after deployment.

The default address will look similar to:

`https://YOUR-GITHUB-NAME.github.io/glrp-website/`

## Optional custom domain

Hosting remains free, but a custom domain such as `gangliferp.co.uk` normally costs money to register. You can connect one later without rebuilding the site.
